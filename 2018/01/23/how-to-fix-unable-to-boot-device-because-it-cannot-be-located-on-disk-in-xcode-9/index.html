修复Xcode9中的“Unable to Boot Device Because It Cannot Be Located on Disk”错误

23 January 2018 in categories

起因

因为手残,把模拟器玩坏了,然后就喜闻乐见的运行不起来了.emmmmm.阴吹斯廷.

清理旧的缓存数据

首先把模拟器程序的的后台服务干掉

sudo killall -9 com.apple.CoreSimulator.CoreSimulatorService

然后删除模拟器相关的文件缓存

rm -rf ~/Library/Developer/CoreSimulator
rm -rf ~/Library/Developer/XCPGDevices

使用snapshot重置模拟器

首先安装snapshot

sudo gem install snapshot

然后重置模拟器

sudo fastlane snapshot reset_simulators

搞定收工.
